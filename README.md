# Feature Flag Service (Lite)

Deploys a single, free "Lite" plan instance of the IBM Cloud App Configuration service.

## Overview

This deployable architecture provisions one instance of the App Configuration service on the "Lite" plan. This service provides a full UI and SDKs (similar to LaunchDarkly) for managing feature flags, user segmentation, and controlled rollouts for your applications.

**This is a free service.** It uses the "Lite" plan, which is not intended for production use but is perfect for development, testing, and learning.
